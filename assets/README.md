# Assets Directory

This directory contains static assets for the NEBB Calculator app.

## Required Files

### App Icons
- `icon.png` - Main app icon (1024x1024 recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (32x32)

### Splash Screen
- `splash.png` - App splash screen (1242x2436 recommended)

## Icon Specifications

### App Icon (icon.png)
- **Size**: 1024x1024 pixels
- **Format**: PNG with transparency
- **Design**: Should represent NEBB calculator functionality
- **Colors**: Use NEBB brand colors (#1976D2, #FF6B35)

### Adaptive Icon (adaptive-icon.png)
- **Size**: 1024x1024 pixels
- **Format**: PNG with transparency
- **Design**: Same as main icon but optimized for Android adaptive icons

### Favicon (favicon.png)
- **Size**: 32x32 pixels
- **Format**: PNG
- **Design**: Simplified version of main icon

### Splash Screen (splash.png)
- **Size**: 1242x2436 pixels (iPhone X resolution)
- **Format**: PNG
- **Design**: NEBB Calculator branding with app name
- **Background**: White (#FFFFFF)

## Placeholder Icons

Until you create custom icons, you can use placeholder icons from:
- [Expo Icons](https://expo.github.io/vector-icons/)
- [Material Design Icons](https://material.io/icons/)
- [Ionicons](https://ionicons.com/)

## NEBB Branding Guidelines

When creating custom icons and splash screens, follow these guidelines:
- Use NEBB blue (#1976D2) as primary color
- Use NEBB orange (#FF6B35) as accent color
- Maintain professional, clean design
- Ensure good contrast for accessibility
- Test on both light and dark backgrounds
